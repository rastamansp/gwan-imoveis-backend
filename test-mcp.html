Gere comandos prontos para testar o servidor MCP (stdio) a partir do seu terminal (cmd/PowerShell). Esta página não executa MCP no navegador; ela auxilia na construção das chamadas JSON‑RPC.

Requisitos: API ativa em MCP_BASE_URL (padrão http://localhost:3001) e servidor MCP via npm run start:mcp:stdio. Se usar autenticação, defina MCP_AUTH_TOKEN e informe o token abaixo.

O que está por trás

  • O servidor MCP é iniciado via npm run start:mcp:stdio e converte operações anotadas com x-mcp.enabled=true (Swagger) em tools.
  • Os métodos expostos atuais: list_events, get_event_by_id, get_event_ticket_categories.
  • Esta página apenas monta a mensagem JSON‑RPC para tools/list e tools/call e a injeta em um comando de terminal.
  • Para integração detalhada com clientes (Claude, VS Code), consulte docs/mcp/integration.md.
