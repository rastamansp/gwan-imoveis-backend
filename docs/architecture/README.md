## Princípios Arquiteturais

O projeto segue os seguintes princípios:

1. **Clean Architecture** - Separação de camadas
2. **SOLID** - Princípios de design
3. **Domain-Driven Design** - Foco no domínio
4. **TOGAF** - Framework de arquitetura empresarial
